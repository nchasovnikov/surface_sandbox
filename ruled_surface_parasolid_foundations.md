# Mathematical Foundations of a Ruled Surface with Two 3D Curves and a Spine Curve (as in Siemens NX / Parasolid)

## 1. Classical Ruled Surface Definition

A ruled surface is generated by moving a straight line (the **ruling**)
along curves.

\[ S(u,v) = C_1(u) + vD(u) \]

where

-   (C_1(u)) = first space curve\
-   (D(u)) = ruling direction vector\
-   \(v\) = parameter along the ruling

If the second curve is (C_2(u)):

\[ D(u) = C_2(u) - C_1(u) \]

Thus:

\[ S(u,v) = (1-v)C_1(u) + vC_2(u) \]

with

\[ u `\in [0,1]`{=tex}, `\quad `{=tex}v `\in [0,1]`{=tex}\]

This represents **linear interpolation between two curves**.

------------------------------------------------------------------------

# 2. Curve Correspondence Problem

Real CAD curves typically have:

-   different parameterizations
-   different arc lengths
-   non-uniform curvature

Therefore mapping functions are introduced:

\[ u_1 = f_1(t), `\quad `{=tex}u_2 = f_2(t) \]

Surface becomes:

\[ S(t,v) = (1-v)C_1(f_1(t)) + vC_2(f_2(t)) \]

------------------------------------------------------------------------

# 3. Spine Curve Control

Let the spine curve be:

\[ S_p(t) \]

The spine determines correspondence between the curves.

Mappings:

\[ u_1 = f_1(t) \]

\[ u_2 = f_2(t) \]

Final surface:

\[ R(t,v) = (1-v)C_1(f_1(t)) + vC_2(f_2(t)) \]

------------------------------------------------------------------------

# 4. Geometric Interpretation

For each spine parameter (t):

1.  Find point on curve 1
2.  Find corresponding point on curve 2
3.  Connect them with a straight line

Surface:

\[ R(t,v) = P_1(t) + v(P_2(t) - P_1(t)) \]

with

\[ P_1(t) = C_1(f_1(t)) \]

\[ P_2(t) = C_2(f_2(t)) \]

------------------------------------------------------------------------

# 5. NURBS Representation

Curves in CAD are typically **NURBS**.

\[ C(u) = `\frac{\sum N_i(u) w_i P_i}{\sum N_i(u) w_i}`{=tex} \]

Ruled surfaces become **tensor‑product NURBS surfaces**.

Control point interpolation:

\[ P\_{i,j} = (1-v_j)P_i\^{(1)} + v_jP_i\^{(2)} \]

------------------------------------------------------------------------

# 6. Differential Geometry

Define:

\[ d(t) = P_2(t) - P_1(t) \]

Surface:

\[ R(t,v) = P_1(t) + v d(t) \]

Derivatives:

\[ R_t = P_1'(t) + v d'(t) \]

\[ R_v = d(t) \]

Normal vector:

\[ N = R_t `\times `{=tex}R_v \]

Gaussian curvature satisfies

\[ K `\le 0`{=tex} \]

Meaning ruled surfaces are either

-   developable
-   saddle shaped

Special cases:

  Surface                 Condition
  ----------------------- -------------------
  Cylinder                (d'(t)=0)
  Cone                    rulings intersect
  Tangent developable     (d(t)=C'(t))
  Hyperbolic paraboloid   doubly ruled

------------------------------------------------------------------------

# 7. Parasolid Kernel Construction Pipeline

Typical algorithm:

## Step 1 --- Input Curves

\[ C_1(u), `\quad `{=tex}C_2(v) \]

Optional spine

\[ S(t) \]

------------------------------------------------------------------------

## Step 2 --- Parameter Normalization

Domains

\[ u `\in [u_a,u_b]`{=tex}, `\quad `{=tex}v `\in [v_a,v_b]`{=tex}\]

mapped to

\[ t `\in [0,1]`{=tex}\]

------------------------------------------------------------------------

## Step 3 --- Curve Compatibility

Ensure identical representation.

### Degree elevation

\[ p_1 = p_2 = p \]

### Knot vector merging

\[ U = U_1 `\cup `{=tex}U_2 \]

------------------------------------------------------------------------

## Step 4 --- Spine Mapping

Closest points are solved:

\[ u_i = `\arg`{=tex}`\min `{=tex}\|\|C_1(u) - S(t_i)\|\|\^2 \]

\[ v_i = `\arg`{=tex}`\min `{=tex}\|\|C_2(v) - S(t_i)\|\|\^2 \]

Mappings fitted as splines

\[ u = f_1(t) \]

\[ v = f_2(t) \]

------------------------------------------------------------------------

# 8. Ruling Construction

Points

\[ P_1(t) = C_1(f_1(t)) \]

\[ P_2(t) = C_2(f_2(t)) \]

Ruling line

\[ L(t,`\lambda`{=tex}) = (1-`\lambda`{=tex})P_1(t) +
`\lambda `{=tex}P_2(t) \]

\[ `\lambda `{=tex}`\in [0,1]`{=tex}\]

------------------------------------------------------------------------

# 9. NURBS Surface Storage

Surface form

\[
S(u,v)=`\frac{\sum_{i,j}N_{i,p}(u)M_{j,q}(v)w_{ij}P_{ij}}{\sum_{i,j}N_{i,p}(u)M_{j,q}(v)w_{ij}}`{=tex}
\]

For ruled surfaces:

-   degree (p) in curve direction
-   degree (1) in ruling direction

Control points

\[ P\_{i,0}=P\_{1,i} \]

\[ P\_{i,1}=P\_{2,i} \]

Weights

\[ w\_{i,0}=w\_{1,i}, `\quad `{=tex}w\_{i,1}=w\_{2,i} \]

Knot vectors

\[ U = `\text{curve knot vector}`{=tex} \]

\[ V = {0,0,1,1} \]

------------------------------------------------------------------------

# 10. Orientation and Twist Check

Tangents

\[ T_1=C_1'(u) \]

\[ T_2=C_2'(v) \]

If

\[ (T_1 `\times `{=tex}d)`\cdot`{=tex}(T_2 `\times `{=tex}d) \< 0 \]

the kernel may reverse one curve to prevent twisting.

------------------------------------------------------------------------

# 11. Surface Validity Checks

### Degeneracy

\[ \|\|P_2(t) - P_1(t)\|\| `\approx 0`{=tex} \]

### Self‑intersection

Checked via bounding volume tests.

### Model tolerance

Typical CAD tolerance

\[ 10\^{-6} \]

------------------------------------------------------------------------

# 12. Final Surface Equation

Without spine

\[ S(u,v) = (1-v)C_1(u) + vC_2(u) \]

With spine

\[ S(t,v) = (1-v)C_1(f_1(t)) + vC_2(f_2(t)) \]

Stored internally as a **degree (p,1) NURBS surface**.

------------------------------------------------------------------------

**Summary:**

A ruled surface in a CAD kernel like Parasolid is essentially a **NURBS
curve extruded linearly to another curve**, optionally with a
spine‑controlled parameter mapping.
